

# Decimal to IEEE-754 转换工具

一个基于 Qt 的图形界面工具，用于将十进制小数转换为 IEEE 754 标准的浮点数表示（单精度 Float32 和双精度 Float64）。

## 功能特性

- **单精度浮点数 (Float32)** 转换：32 位 IEEE 754 单精度格式
- **双精度浮点数 (Float64)** 转换：64 位 IEEE 754 双精度格式
- 支持二进制和十六进制输出格式
- 图形用户界面，操作简单直观

## 项目结构

```
├── CMakeLists.txt       # CMake 构建配置文件
├── convertlogic.cpp     # 转换逻辑实现
├── convertlogic.h       # 转换逻辑头文件
├── main.cpp             # 程序入口
├── mainwindow.cpp       # 主窗口实现
└── mainwindow.h         # 主窗口头文件
```

## 核心技术

- **Qt**：图形界面框架
- **C++**：编程语言
- **IEEE 754**：浮点数表示标准

## 依赖

- Qt 5.x 或更高版本
- C++11 标准兼容的编译器

## 构建方法

```bash
# 使用 CMake 构建
mkdir build
cd build
cmake ..
make
```

## 使用方法

1. 运行程序
2. 选择转换类型（单精度/双精度）
3. 输入十进制数值
4. 点击计算按钮
5. 查看结果（二进制和十六进制表示）

## 转换逻辑说明

程序将十进制浮点数转换为 IEEE 754 格式：
- **Float32 (单精度)**：1 位符号 + 8 位指数 + 23 位尾数
- **Float64 (双精度)**：1 位符号 + 11 位指数 + 52 位尾数
